# 동아리 관리 프로그램

## 1. 개요
### 1.1 목적
동아리 정보를 체계적으로 관리하고 활동 보고서를 작성하여 대학 내 동아리 운영을 지원합니다.

### 1.2 대상
- 청주대 동아리 운영진
- 청주대 지도교수
- 청주대 학생지원팀

## 2. 프로그램의 중요성 및 필요성
### 2.1 중요성
- 동아리 운영의 투명성과 효율성 향상.
- 활동 내역 기록 자동화.

### 2.2 필요성
- 데이터 손실 방지.
- 동아리 활동 평가 및 지원 간소화.
